# Exploratory Data Analysis (EDA) on Layoffs Dataset

## Dataset
Layoffs dataset containing company-wise, industry-wise and country-wise layoff information.

## Objective
To explore the dataset and identify patterns, trends and insights related to layoffs.

## Analysis Performed
- Year-wise layoff trends
- Industry-wise and country-wise analysis
- Company-level layoff impact
- Funding stage vs layoffs analysis

## Tools Used
- SQL
- MySql

## Key Insights
- Layoffs peaked during specific years
- Certain industries were more impacted
- Large funding did not prevent layoffs

## Dataset Source
- www.kaggle.com

## Data Cleaning:
- This analysis was performed on cleaned data.
- Data cleaning was done in a separate SQL project:https://github.com/JainSani07/sql-data-cleaning-project

 This project is part of an end-to-end SQL data analysis workflow,including data cleaning and exploratory analysis.
